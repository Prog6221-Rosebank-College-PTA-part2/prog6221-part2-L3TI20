# CyberBot — Cybersecurity Awareness Assistant (Part 2)

A modern WinForms desktop chatbot for cybersecurity education.
Purple/indigo themed, two-panel layout with sidebar topic shortcuts.

## Features
- Voice greeting on startup
- Sidebar with 12 quick-access topic buttons
- Rounded chat bubbles (user right, bot left)
- 15+ cybersecurity topics with random response variants
- Memory — tracks your name, interests, and message count
- Sentiment detection — adapts tone to your mood
- Follow-up flow — type "tell me more" to expand any topic

## How to Run
1. Open `CyberSecurityBotPart2.csproj` in Visual Studio 2022
2. Add `voiceGreeting.wav` → set Copy to Output Directory → Copy if newer
3. Press F5

## Requirements
.NET 8.0 · Windows · Visual Studio 2022
